Welcome to my first website!


This website is where I practice CSS styling and JavaScript, using no pre-configured code. I want everything on this website to be coded by hand, regardless of complexity. This is also where I will practice using modern web technologies, such as CSS pre-compilation, and frameworks.

With these things in mind, this website will likely never be made to look very attractive, but rather be the ugly-duckling testing grounds for concepts I want to practice.

TLDR: This ain't no Bootstrap site, kiddo.
